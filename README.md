# wlfx
WLFX is a JavaFX-based Weblate client

Copyright (C) 2020-2022 Javier Llorente <javier@opensuse.org>


Dependencies
------------
* JavaFX 11+
* jakarta.json
* jgettext (from https://github.com/javierllorente/jgettext)
* ikonli-javafx
* ikonli-icomoon-pack
* java-diff-utils

Build & Run with Maven
------------
`mvn javafx:run`
