# wlfx
WLFX is a JavaFX-based Weblate client


Dependencies
------------
* JavaFX 11+
* jwl (from https://github.com/javierllorente/jwl)
* jgettext (from https://github.com/javierllorente/jgettext)
* ikonli-javafx
* ikonli-icomoon-pack
* java-diff-utils

Build & Run with Maven
------------
`mvn javafx:run`
